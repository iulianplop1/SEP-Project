# Cloverville

An eco-village community management platform built during my first-semester software engineering project (SEP1).

## Why We Built This
For our SEP1 project, my team and I were tasked with building a complex management system from scratch. We decided to build Cloverville: a sustainability-focused community platform. 

The application allows "villagers" to track their green activities, set community sustainability goals, manage shared tasks, and even operate a local trading system for resources. It was a massive learning experience in object-oriented programming, MVC architecture, and working collaboratively in a team.

## Features
- **Green Activities & Goals**: Track personal and community sustainability efforts.
- **Shared Tasks**: Distribute and manage community workloads.
- **Trading System**: An internal economy for villagers to trade resources.
- **Village Management**: Comprehensive tools for managing villagers and village data.
